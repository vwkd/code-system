# Comment



## Introduction

- can be line, block or documentation



## Line

- starts with two forward slashes
- single-line



## Block

- starts with forward slash and star
- ends with star and forward slash
- multi-line



## Documentation

- starts with three forward slashes
- single-line
- above the item
- supports markdown
- code blocks are tested
