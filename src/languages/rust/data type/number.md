# Number



## Introduction

- scalar type



## Integer

- unsigned or signed

### unsigned

- non-negative
- `u8`: 8-bit
- `u16`: 16-bit
- `u32`: 32-bit
- `u64`: 64-bit
- `u128`: 128-bit
- `usize`: pointer-sized, platform-specific

### signed

- `i8`: 8-bit
- `i16`: 16-bit
- `i32`: 32-bit
- `i64`: 64-bit
- `i128`: 128-bit
- `isize`: pointer-sized, platform-specific



## Float

- rational number
- `f32`: 32-bit
- `f64`: 64-bit
